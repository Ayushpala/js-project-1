# JavaScript / TypeScript Mini Projects 🚀

This repository contains beginner-friendly JavaScript/TypeScript programs created for logic building and basic programming practice.

---

# 📚 Programs List

## 1. Age Finder

Calculates the age of a person using birth year and current year.

### Formula:

```js
Age = Current Year - Birth Year
```

---

## 2. BMI Calculator

Calculates Body Mass Index (BMI).

### Formula:

```js
BMI = Weight / (Height × Height)
```

---

## 3. Calculator

Performs basic arithmetic operations:

## 4. Grade Calculator

Checks grade based on marks using if-else conditions.

### Example:

* 90+ → Grade A
* 75–89 → Grade B
* 50–74 → Grade C

---

## 5. Largest Number Finder

Finds the largest number among three numbers.

---

## 6. Leap Year Validator

Checks whether a year is a leap year or not.

### Logic:

```js
Year % 4 == 0
Year % 100 != 0
OR
Year % 400 == 0
```

## 12. Simple Interest Calculator

Calculates Simple Interest.

### Formula:

```js
SI = (P × R × T) / 100
```

Where:

* P = Principal Amount
* R = Rate of Interest
* T = Time

## 16. Celsius To Fahrenheit Converter

Converts Celsius temperature into Fahrenheit.

### Formula:

